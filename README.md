### Пошаговая инструкция

Команды написаны для bash  

Создать виртуальное окружение  

```
python -m venv venv
```
Активировать виртуальное окружение
```
source venv/Script/activate
```
Установить зависимости
```
pip install -r requirements.txt
```
Запустить app.py
